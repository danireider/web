Ejemplos de HTML

HTML Introduction

HTML significa HyperText Markup Language y se usa para estructurar páginas web.

HTML Editors

Puedes escribir HTML en cualquier editor de texto como Notepad++ o VS Code.

HTML Basic

Ejemplo de una estructura básica en HTML:

        <!DOCTYPE html>
        <html>
        <head>
            <title>Mi página</title>
        </head>
        <body>
            <p>Hola, mundo!</p>
        </body>
        </html>

HTML Elements

Todo en HTML está compuesto por elementos, como los encabezados <h1>, párrafos <p> y enlaces <a>.
